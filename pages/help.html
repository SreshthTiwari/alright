Can I use this on a Mobile Device?

Yes, this website is fully responsive for both Mobile and Tablet devices. The user will get the same, interactive experience on all kinds of devices.

Do I need to fill in all fields in the form?

No, you only need to fill in the forms of the events for which your school will be participating

How do I report a problem in the website?

If you encounter a bug or problem while using the website, you can take a screenshot or recording of it, and mail it to: litfest@es07.euroschoolindia.com
